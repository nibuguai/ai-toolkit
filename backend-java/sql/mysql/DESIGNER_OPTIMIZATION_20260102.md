# 首页设计器优化更新 - 20260102

## 🎯 更新内容

### 1. 修复属性配置中的Switch开关问题 ✅

**问题描述**：
- 显示标题开关无法切换，一直处于禁用状态
- Switch 组件太长，不美观

**解决方案**：
- 使用 `v-model:checked` 代替 `v-model:value` 来绑定 Switch 的值
- 添加 `size="small"` 属性使开关更紧凑
- 为 boolean 类型单独处理，不再使用动态组件

**代码修改**：
```vue
<!-- 布尔开关 -->
<Switch
  v-if="prop.type === 'boolean'"
  v-model:checked="formData[prop.key]"
  size="small"
  @change="handleUpdateConfig"
/>
```

---

### 2. 添加属性配置面板收起/展开功能 ✅

**功能说明**：
- 右侧配置面板可以收起，增加设计区域空间
- 收起后显示一个小按钮，点击可展开
- 展开时右上角有收起按钮
- 使用 CSS 过渡动画，体验流畅

**实现效果**：
- **展开状态**：宽度 320px（w-80）
- **收起状态**：宽度 48px（w-12），只显示一个展开按钮
- **过渡动画**：300ms 平滑过渡

**使用方法**：
1. 点击配置面板右上角的 ▶ 按钮收起面板
2. 点击收起后左侧的 ◀ 按钮展开面板

---

### 3. 全局布局配置功能 ✅

**功能说明**：
提供全局设置对话框，允许用户动态调整：
- **容器内边距**：控制画布四周的空白区域（0-50px）
- **组件间距**：控制组件之间的间隔距离（0-30px）

**使用方法**：
1. 点击顶部工具栏的"全局设置"按钮
2. 在弹出的对话框中调整参数：
   - **容器内边距**：设置为 0 可让组件紧贴边缘
   - **组件间距**：建议值 5-15px
3. 调整后实时生效
4. 保存布局时会一起保存这些配置

**默认值**：
- 容器内边距：10px
- 组件间距：10px

## 📞 技术说明

### Switch 组件绑定
Ant Design Vue 的 Switch 组件：
- ✅ 正确：`v-model:checked`
- ❌ 错误：`v-model:value`

### 配置存储
全局配置当前存储在布局配置的 JSON 中：
```json
{
  "items": [...],
  "colNum": 24,
  "margin": [10, 10],
  "containerPadding": [10, 10],
  ...
}
```

### 动态样式应用
```vue
<GridLayout
  :margin="[margin, margin]"
  :style="{ padding: `${containerPadding}px` }"
/>
```
