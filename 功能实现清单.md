### Sprint 2.3：系统级实时变声器（Android APP）

> **核心需求：** APP 后台常驻，实时变声，微信/QQ/游戏等其他 APP 可直接使用变声后的声音。
> **技术方案：** Android 原生 NDK（C++） + SoundTouch + 虚拟音频设备
> **平台要求：** Android 11+，不需要 root

#### 2.3.1 变声引擎（C++ NDK 层）

- [ ] **集成 SoundTouch 库**
  - 音高变换（Pitch Shift）：-12 ~ +12 半音
  - 速率调整（Tempo）：0.5x ~ 2.0x
  - 延迟目标：< 10ms 单帧处理
- [ ] **DSP 处理管线**
  - AudioRecord 采集（16kHz/44.1kHz，16bit，单声道）
  - 预加重滤波 → SoundTouch 音高变换 → 共振峰微调 → 后处理（EQ/混响/压缩）
  - AudioTrack 输出到虚拟音频设备
- [ ] **JNI 接口**
  - `nativeInit()` — 初始化引擎
  - `nativeSetPreset(int presetId)` — 切换预设
  - `nativeSetPitch(double semitones)` — 手动调音高
  - `nativeSetEffect(int effectType, float value)` — 设置效果器参数
  - `nativeProcess(byte[] input)` — 处理一帧音频
  - `nativeRelease()` — 释放资源

#### 2.3.2 预设音色（8 种基础 + 自定义）

- [ ] **基础预设**

| 预设名 | 音高 | 其他效果 | 适用场景 |
|--------|------|----------|----------|
| 男声 | -5 半音 | 共振峰微调 | 女变男 |
| 女声 | +7 半音 | 共振峰调整 | 男变女 |
| 萝莉 | +12 半音 | 气声增强 | 变声搞怪 |
| 大叔 | -8 半音 | 低频增强 | 沉稳音色 |
| 机器人 | 0 | 量化 + 环形调制 | 科幻效果 |
| 花栗鼠 | +12 半音 | 速率加快 | 搞笑效果 |
| 巨人 | -12 半音 | 混响 + 低频 | 震撼效果 |
| 空灵 | +3 半音 | 大混响 + 延迟 | 仙气效果 |

- [ ] **自定义调节**
  - 音高滑块（-12 ~ +12 半音）
  - 混响强度（0 ~ 100%）
  - 语速（0.5x ~ 2.0x）
  - 保存自定义预设

#### 2.3.3 Android APP 架构

- [ ] **前台服务（Foreground Service）**
  - 通知栏常驻："变声中 · 当前：萝莉音"
  - 点击通知打开控制面板
  - 系统不会杀掉前台服务
- [ ] **悬浮窗控制面板**
  - 小悬浮球：显示变声状态，点击展开
  - 展开面板：预设切换、音高滑块、开关按钮
  - 全局可用，不遮挡其他 APP
- [ ] **虚拟音频设备**
  - 使用 Android AudioPlaybackCapture API（Android 11+）
  - MediaProjection 授权 → 创建虚拟麦克风
  - 其他 APP 设置声音输入为虚拟设备
- [ ] **权限管理**
  - 麦克风权限（RECORD_AUDIO）
  - 前台服务权限（FOREGROUND_SERVICE）
  - 悬浮窗权限（SYSTEM_ALERT_WINDOW）
  - MediaProjection 授权（弹窗确认）

#### 2.3.4 用户使用流程

```
1. 安装 APP → 授权麦克风 + 悬浮窗 + MediaProjection
2. 打开 APP → 选择变声效果（萝莉/大叔/机器人等）
3. 点击"开启变声" → APP 最小化到后台，悬浮球显示
4. 打开微信/QQ/游戏 → 设置声音输入为"AI Toolkit 变声器"
5. 说话 → 对方听到的是变声后的声音
6. 点击悬浮球 → 随时切换效果或关闭变声
```

#### 2.3.5 服务端接口（可选，后期扩展）

- [ ] **预设同步** `GET /api/v1/language/voice-changer/presets`
  - 从服务器拉取预设列表（支持在线更新新预设）
- [ ] **AI 音色转换** `POST /api/v1/language/voice-changer/ai-convert`
  - 上传录音 → 服务端 AI 模型处理 → 返回高质量变声音频
  - 用于对质量要求高的场景（非实时）
- [ ] **用户预设云同步**
  - 自定义预设保存到云端，多设备同步

#### 2.3.6 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| Android 框架 | Kotlin + Jetpack Compose | UI 层 |
| 音频采集 | AudioRecord / Oboe (NDK) | 低延迟采集 |
| 变声引擎 | SoundTouch (C++) | 开源，延迟 < 10ms |
| 效果器 | 自研 DSP (C++) | 混响/EQ/压缩 |
| 虚拟设备 | MediaProjection + AudioPlaybackCapture | Android 11+ |
| 悬浮窗 | WindowManager + Compose | 全局控制 |
| 构建工具 | Gradle + CMake (NDK) | C++ 编译 |
