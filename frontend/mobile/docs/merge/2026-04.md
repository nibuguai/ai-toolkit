## `1df22539` 分析

上游改动：

- `vite.config.ts` 引入 `isMpWeixin`。
- 将 `UniOptimization({ enable: { optimization, async-import, async-component } })` 改为 `UniOptimization({ enable: isMpWeixin })`。
- 目的：仅在微信小程序端开启 `optimization`，修复运行到支付宝小程序时报错的问题。

本地处理：

- 不额外引入 `@uni-helper/uni-env` 的 `isMpWeixin`。
- 复用当前 `vite.config.ts` 已有的 `UNI_PLATFORM`，改为：

```ts
enable: UNI_PLATFORM === 'mp-weixin'
```

处理原因：

- 当前配置中已经统一用 `UNI_PLATFORM` 判断 H5、App、小程序等平台。
- 表达的条件与上游 `isMpWeixin` 等价。
- 避免为单个布尔值增加新的 import 风格。
